Forms ​

Tachyons has some basic form control resets to remove default styles for mobile devices.

Example ​

vue
<template>
  <form>
    <label for="inputReset">
      <code v-f="6" v-db>input-reset</code>
    </label>
    <input id="inputReset" v-input-reset type="text">
  </form>
</template>
