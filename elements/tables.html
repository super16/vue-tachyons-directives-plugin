Tables #

Tachyons has some basic utilities for styling tables that are easy to customize and extend.

Examples #

Striped | Light Gray #

<table v-collapse>
  <tbody>
    <tr v-striped:light-gray></tr>
    <tr v-striped:light-gray></tr>
  </tbody>
</table>

Striped | Near white #

<table v-collapse>
  <tbody>
    <tr v-striped:near-white></tr>
