Text Decoration #

Utilities for decorating text with underlines, or removing the default underlines browsers put on links.

Examples #

None #

<a v-no-underline href="#">no-underline</a>
no-underline

Underline #

<p v-underline>underline</p>

underline

Strikethrough #

<p v-strike>strikethrough</p>

strikethrough
